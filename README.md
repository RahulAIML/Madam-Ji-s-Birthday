# A Wish For My Love

A single-page, romantic, minimalist website to celebrate Mou. Built with HTML, CSS, and vanilla JS.

## How to Run
- Simply open `index.html` in your browser (double-click or drag into a tab).

## Customize
- Replace `[Image Placeholder URL]` in `index.html` with your actual image URLs for the 6 gallery images and the secret image.
- The greeting currently types: `Happy Birthday, Mou!` â€” update the name inside `script.js` if needed.
- The unlock code is `0123`. Change `SECRET_CODE` in `script.js` to update it.
- All colors and typography are set in `styles.css`.

## Features
- Full-screen hero with elegant script headline and smooth typing animation.
- Responsive memories gallery (6 images) with glassy cards.
- Heart FAB button that spawns floating hearts.
- Unlockable surprise section that fades in upon correct code.

## Tech
- Fonts: Google Fonts `Great Vibes`, `Poppins`, and `Montserrat`.
- Colors: Soft Pink `#FFB6C1`, Deep Purple `#4B0082`, White `#FFFFFF`.
